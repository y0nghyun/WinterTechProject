# WinterTechProject
**레디스, 카프카, 도커, e2c, postgresql 과 같은 기술 사용에 집중해 보는 프로젝트**

| **주간 Workflow**

* (월요일 오후) 1차 작업 pr(각자의 main branch에 대해)
* (수요일 밤) 작업 pr에 대해 후에 기한 수요일 밤까지 review 진행하기
* (금요일 오전) review 받은 코드에 대해 리팩토링 후 2차 pr
* (금요일 오전) 금요일 아침에 보면서 review + merge
